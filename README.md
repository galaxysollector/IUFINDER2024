이 프로그램은 64비트의 Windows 10/11만을 지원합니다. **이외의 운영체제는 지원하지 않습니다!**

[이곳](https://github.com/galaxysollector/IUFINDER2024/releases/latest)으로 이동하여 최신 버전을 다운로드한 후 프로그램의 지시를 따르십시오.
